# Mason Sand

## Summary
Mason sand is a fine, clean sand. It is free of rocks and debris.

## Key Details
- **Texture**: Fine, smooth.
- **Uses**: Mixing mortar for bricklaying, under pool liners, sandboxes, beach areas.

## When This Applies
- "I'm putting up a swimming pool."
- "I need sand for a sandbox."

## How The Agent Should Respond
"For a pool or sandbox, you want Mason Sand. It's the finest, smoothest sand we have."

## Red Flags / Misconceptions
- Don't use for paver base (use Concrete Sand for that, as it drains better).

## Internal Notes (Not customer-facing)
- Often called "Pool Sand".
