# Fill Dirt

## Summary
Fill dirt is unscreened, raw dirt. It may contain rocks, clay, and clumps. It is used for filling deep holes where quality doesn't matter.

## Key Details
- **Type**: Unscreened / Raw.
- **Uses**: Filling deep holes, building up grade, backfilling foundations.

## When This Applies
- "I have a giant hole to fill."
- "I need the cheapest dirt you have."
