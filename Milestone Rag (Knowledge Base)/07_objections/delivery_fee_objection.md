# Objection: Delivery Fee

## Summary
Customer thinks delivery is too expensive.

## Key Details
- **Response**: Explain Distance Matrix (fairness).
- **Value**: Saves them time/gas/rental.

## How The Agent Should Respond
"I understand. We use a system that calculates the exact mileage so you only pay for the distance we travel. It's actually cheaper than a flat rate for most folks nearby."
