# Objection: Dumping Location

## Summary
Customer insists on a specific (unsafe) dump spot.

## Key Details
- **Safety First**: We can't risk wires or tipping.
- **Driver Authority**: Driver decides.

## How The Agent Should Respond
"I can put that in the notes, but ultimately the driver has to make the call when they see it. If it's not safe, they'll have to put it somewhere else."
