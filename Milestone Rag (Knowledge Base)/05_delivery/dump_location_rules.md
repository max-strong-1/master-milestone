# Dump Location Rules

## Summary
Where we can and cannot dump.

## Key Details
- **Hard Surface**: Preferred (Driveway).
- **No Overhead**: Clear of wires.
- **Driver Discretion**: Driver has final say.

## How The Agent Should Respond
"We usually dump on the driveway or a hard surface. The driver will check the spot when they arrive to make sure it's safe."
